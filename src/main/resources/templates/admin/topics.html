Импорт тем из JSON

Формат загружает дерево тем для одного предмета и класса. Импорт атомарный: если один узел невалиден, ничего не сохранится. Дубли в одной ветке по code перезаписываются.

POST /api/admin/topics/import
Поля JSON subjectCode или subjectId, gradeNo или gradeId, importNote, массив topics. У каждой темы: code, titleRu, titleKk, опционально children. 
{
  "subjectCode": "math",
  "gradeNo": 4,
  "importNote": "Математика 4 класс, книга 2026",
  "topics": [
    {
      "code": "numbers-and-values",
      "titleRu": "Числа и величины",
      "titleKk": "Сандар және шамалар",
      "children": [
        {
          "code": "multi-digit-numbers",
          "titleRu": "Многозначные числа",
          "titleKk": "Көптаңбалы сандар"
        }
      ]
    }
  ]
}
