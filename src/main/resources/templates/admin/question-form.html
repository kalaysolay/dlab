SCQ / MCQ варианты

A

Для SCQ сервер требует ровно один правильный вариант; для MCQ - минимум один.

MATCHING пары

Ключ сопоставления хранится по строкам: левая часть строки соответствует правой части той же строки.

FILL_IN ключ ответа

Как составлять FILL_IN: 1) В тексте вопроса используйте метки вида [[1]], [[2]]. 2) Ниже добавьте для каждой метки правильный ответ и правило проверки. 3) Для NUMERIC_TOLERANCE обязательно укажите tolerance (например 0.01). 4) Если в тексте есть [[1]], ниже должна быть строка с точно таким же placeholder.

Пустые строки не сохраняются. Для каждой метки из текста вопроса должна быть заполнена строка с тем же placeholder.
