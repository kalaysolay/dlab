Flow: запуск предметного среза

Предметный срез
Шаг 1 Тип Выбор категории теста.
Шаг 2 Параметры Фильтры: предмет -> класс -> язык.
Шаг 3 Инициализация Подбор вопросов и запуск сессии.
Ошибка

Сессия создается как уникальный SessionID с вопросами по выбранным параметрам. Поддерживаемые типы: SCQ, MCQ, MATCHING, FILL_IN.
